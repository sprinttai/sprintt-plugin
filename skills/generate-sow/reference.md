# Reference — Generate SOW

This skill reads live data from the private **knowledge-base** repository via the GitHub connector.
No business data is stored here.

## KB Dependencies

| File | Purpose |
|------|---------|
| `legal/templates/sow.md` | SOW template with all sections and bracketed fields |
| `services/pricing.md` | Pricing ranges, payment structure rules, and hourly rates |
| `services/offerings.md` | Service descriptions for accurate scope articulation |
| `company/entity.md` | Sprintt entity name, address, and signatory |
